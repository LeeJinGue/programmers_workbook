## 프로그래머스 - 주식가격 문제
## 문제설명

- 초 단위로 기록된 주식 가격이 담긴 배열이 있을 때 가격이 떨어지지 않은 기간은 몇 초인지를 return합니다.

## 변수 설명
- prices: 초 단위로 주식 가격이 담긴 변수(int[])
- answer: 가격이 떨어지지 않은 기간을 담은 변수(int[])

## 제한사항
- prices의 각 가격은 1 ~ 10000 입니다.
- prices의 길이는 2 ~ 100000 입니다.
